    亚洲、欧洲、北美洲
    中国东北地区杨树的毁灭性害虫。幼虫在木质部和韧皮部之间环绕树干蛀食为害，导致枯死或遇风折断。被害率有时可达100%。东北地区是我国重要的木材生产基地，一旦传入我国并定殖，将对我国的林业、生态、环境和旅游资源造成巨大损失，甚至是毁灭性的危害。
